# calendario-DWYM

Historia 1: Programar un evento
Como usuario,
quiero programar que día tengo un evento,
para organizarme la semana.
Criterios de aceptación:
Se muestran los días del mes/semana/día.
Se muestran los eventos anteriormente programados.
Se permite escribir un título, fecha, hora de inicio y hora de fin.
Una vez se registra un evento la vista se actualiza automáticamente.

Historia 2: Eliminar un evento
Como usuario,
debo poder borrar un evento si se cancela,
para no confundirme.
Criterios de aceptación:
Se muestran los eventos agendados en el calendario
aparece un botón que permita eliminar el evento al editarlo	
Se muestra un loader mientras carga
el evento no aparece en el calendario 

Historia 3: Modificar evento 
Como usuario,
quiero poder modificar un evento ya creado,
para corregir si hay algún dato erróneo.
Criterios de aceptación:
Se muestran los eventos agendados en el calendario
aparece un botón que permita modificar el evento 	
Se muestra un loader mientras carga
El evento es modificado según el parámetro
Si se le modifica la fecha al evento, automáticamente cambia de lugar.

Historia 4: Ver evento 
Como usuario,
quiero poder ver un evento registrado,
para ver lo que tengo anotado sobre ese evento.
Criterios de aceptación:
Se muestran los eventos agendados en el calendario.
aparece un botón que permite ver en profundidad el evento. 	
Se muestra un loader mientras carga.
Se muestra toda la información del evento.
